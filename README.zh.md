# xs-chat-frontend

一个实现群聊和私聊的Web应用程序。

> 后端: https://github.com/FishCharlotte/xs-chat-backend

## 简介

xs-chat-frontend 是一个基于React的前端项目，旨在提供一个实时聊天平台，支持群聊和私聊功能。

## 功能

- 实时群聊
- 实时私聊
- 本地历史消息
- 表情
- 消息通知
- 好友管理

## 待开发

- 语音和视频通话 (WebRTC)
- 大文件传输
- 移动端适配 (PWA)
- 多语言支持
- 主题切换

## 安装

请确保您已安装 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)。

```bash
git clone https://github.com/FishCharlotte/xs-chat-frontend.git
cd xs-chat-frontend
npm install
```

## 使用

在本地启动开发服务器：

```bash
npm start
```

构建生产版本：

```bash
npm run build
```

## 技术栈

- React
- Redux
- React Router
- Tailwind CSS
- Socket.io
- Axios
